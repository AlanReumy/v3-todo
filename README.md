# v3-todo

**v3-todo** 是基于Vue 3 + Typescript + Vite开发的 web端todo应用

## 使用方法

clone:

```shell
git clone git@github.com:ZimTang/v3-todo.git
```

install:

使用npm

```shell
npm install
or
yarn
```

运行项目:

```shell
npm run dev
or
yarn run dev
```

## 目前完成的功能有

- [x] todo的增加与删除
- [ ] todo的分组
- [x] todo的本地保存
- [ ] 用户登录注册功能
- [ ] todo的在线保存
